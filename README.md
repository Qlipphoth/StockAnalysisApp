# StockAnalysisApp

这是一个用于股票数据分析与结果记录的项目。包含数据、分析脚本、以及生成的报告与图片。

主要目录结构：
- `AnalysisResults/`：分析结果与生成的报告、图片。
- `data/`：原始或处理后的数据文件。
- `Scripts/`：辅助脚本和工具（示例：文档转图片脚本）。

快速开始：
1. 克隆仓库：

   git clone https://github.com/Qlipphoth/StockAnalysisApp.git

2. 进入项目并创建虚拟环境（可选）：

   python -m venv .venv
   .venv\Scripts\activate

3. 安装依赖（如有 `requirements.txt`）：

   pip install -r requirements.txt

4. 运行分析脚本或查看 `AnalysisResults/` 中的报告。

注意：仓库中部分数据目录如 `data/` 和 `logs/` 默认被 `.gitignore` 忽略，敏感文件 `key.md` 也被忽略。
